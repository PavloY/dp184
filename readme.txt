To use this program you must create file 'dev.properties' using path 'src/main/resources/dev.properties' which
 contains the following data: site url, admin site url, path for your driver, your driver.


 Example dev.properties
 =============================================================================
 Site=http\://184-dp.tk
 AdminSite=http\://184-dp.tk/admin
 Driver=webdriver.gecko.driver
 PathForDriver=D\:\\Java\\drivers\\geckodriver.exe

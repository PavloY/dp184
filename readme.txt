To use this program you must create file 'develop.properties' using path 'src/main/resources/develop.properties' which
 contains the following data: site url, admin site url, path for your driver, your driver.


 Example develop.properties
 =============================================================================
Site=http\://184-dp.tk
AdminSite=http\://184-dp.tk/admin
#Driver=remote
PathForDriver=D\:\\Java\\drivers\\geckodriver.exe
Driver=webdriver.gecko.driver
Selenoid=http://144.76.5.68:4444/wd/hub
#UI selenoid link: http://144.76.5.68:6258/#/
